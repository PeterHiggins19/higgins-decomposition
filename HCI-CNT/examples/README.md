# Examples — quickstart + tutorials

Self-contained examples that walk through the engine from data to
report. Run from inside `HCI-CNT/`:

```bash
python examples/01_quickstart.py
python examples/02_custom_dataset.py
python examples/03_render_atlas.py
python examples/04_compare_runs.py
```

| Example | What it does |
|---|---|
| `01_quickstart.py` | Five-minute walkthrough on Japan EMBER reference data |
| `02_custom_dataset.py` | Synthetic 3-carrier composition; demonstrates building input from scratch |
| `03_render_atlas.py` | Batch atlas rendering with run catalog |
| `04_compare_runs.py` | Run-vs-run delta tool walkthrough |

For deeper detail, see
[`../handbook/VOLUME_2_PRACTITIONER_AND_OPERATIONS.md`](../handbook/VOLUME_2_PRACTITIONER_AND_OPERATIONS.md)
Part E (Engine usage).
