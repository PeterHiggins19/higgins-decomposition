# 08_extended — Extended-battery Stage 2 PDFs

The extended-battery experiments rendered as Stage 2 pages: ChemixHub
oxide compositions, ESA Planck cosmic-budget, financial-sector flows,
IIASA NGFS scenarios, and Markham budget. These are the cross-domain
demos that show the engine works on data far outside the EMBER energy
domain it was tuned on.

## Contents

```

├── stage2_chemixhub_oxide.pdf

```

Each PDF is auto-titled with its dataset name + date range, hash-stamped,
and ordered by the doctrine (geometry-first, then dynamics).
