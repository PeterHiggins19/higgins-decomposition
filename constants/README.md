# constants — transcendental constant library

The 35-constant transcendental reference library used by the Super-Squeeze
operator (§8.4 of the foundational Higgins Coordinate System spec).

Each entry is a high-precision rational or transcendental constant with
its name, symbol, decimal expansion, and the structural meaning it
represents (e.g. π, e, φ, Catalan's constant, Glaisher–Kinkelin, etc.).

## Status

This library is preserved as the foundational reference. The CNT engine
(2.0.4 / schema 2.1.0) does not currently invoke transcendental matching
inside the canonical 25-experiment corpus; transcendental matching
remains available as a follow-on track. See
[`../HCI-CNT/handbook/VOLUME_2_PRACTITIONER_AND_OPERATIONS.md`](../HCI-CNT/handbook/VOLUME_2_PRACTITIONER_AND_OPERATIONS.md)
Part H (Hs-Lab integration plan, "constant-library" component) for the
landing path.
