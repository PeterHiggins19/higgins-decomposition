# Hs Papers

Published documents, conference materials, and validation journals for the Higgins Decomposition.

This directory is organised into three subdirectories that reflect the lifecycle of the research: flagship publications, conference engagement, and formal validation.

---

## Directory structure

| Subdirectory | Purpose |
|---|---|
| [flagship/](flagship/) | Main published and presentable documents |
| [codawork2026/](codawork2026/) | CoDaWork 2026 conference materials (Coimbra, Portugal, 1--5 June 2026) |
| [validation/](validation/) | Validation journals and proofs |
