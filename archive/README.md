# Hˢ Archive

This folder holds files that have been superseded by current standards, contain retired terminology that cannot be cleanly migrated (e.g., historical HUF-era documents where the terminology is integral to the content), or are no longer part of the active Hˢ system.

**Policy:** Files are moved here rather than deleted. The archive preserves the intellectual lineage while keeping the active repository clean and coherent.

**Current contents:** Empty — all active documents have been migrated in place to current terminology as of April 29, 2026. No files required archival; all were fixable.

**When to archive a file:**

- It contains retired terminology that is too deeply embedded to fix without destroying historical meaning
- It has been replaced by a newer version and the old version serves no active purpose
- It documents a methodology or approach that has been formally superseded

**When NOT to archive:**

- The file is still actively referenced by other documents or code
- The retired terminology can be replaced in place (fix it instead)
- The file contains historical lineage sections that correctly use retired terms in context
