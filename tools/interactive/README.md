# Hs Interactive Visualizations

Nine standalone HTML files that open directly in any modern browser.
No server, no build step, no dependencies -- each file is entirely
self-contained.

---

## Usage

Open any `.html` file in a browser:

```
open Hs_Spectrum_Analyzer.html        # macOS
xdg-open Hs_Spectrum_Analyzer.html    # Linux
start Hs_Spectrum_Analyzer.html       # Windows
```

Or double-click the file in your file manager.
