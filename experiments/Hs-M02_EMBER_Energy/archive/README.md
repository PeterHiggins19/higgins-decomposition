# Archive — Hs-M02 EMBER Energy

Legacy interactive views from the original Hs 12-step pipeline run on the
EMBER energy dataset (8 countries + World). Kept verbatim for historical
reference.

## Contents

- `Hs-M02_helix.html` — helix viewer
- `Hs-M02_manifold.html` — manifold viewer
- `Hs-M02_pipeline_slideshow.html` — original pipeline-slide deck
- `Hs-M02_projector.html` — original 3D projector
