# Hs-15 hBN Dielectric

Foundational Hs-pipeline experiment record.

This folder contains the input data, full pipeline output, multi-language
diagnostic reports, and visualisations from the original Hs 12-step
pipeline run on this dataset.

| File pattern | Purpose |
|---|---|
| `*.csv` | Source data (where applicable) |
| `Hs-15_hBN_Dielectric_results.json` | Full pipeline output: carriers, CLR coordinates, variance, diagnostics |
| `Hs-15_hBN_Dielectric_report_<lang>.txt` | Diagnostic report in 5 languages (en, pt, it, zh, hi) |
| `pipeline_output/` | Intermediate per-step outputs |
| `JOURNAL.md` | Per-experiment journal (where present) |

## Status

This experiment is part of the **historical Hs-NN series (1–25)** — the
foundational set produced by the original Hs 12-step pipeline. The
current canonical engine is **CNT 2.0.4** at [`../../HCI-CNT/`](../../HCI-CNT/),
and the current canonical experiment corpus (25 experiments at engine
2.0.4 / schema 2.1.0) is at
[`../Hs-CNT_2026-05/`](../Hs-CNT_2026-05/).

This Hs-NN folder is preserved as the original-pipeline record for
lineage. The two series are complementary: the Hs-NN series demonstrates
the original pipeline across 18 physical domains; the Hs-CNT_2026-05
series demonstrates the matured engine on a different 25-experiment
suite focused on the CodaWork 2026 deliverable.
