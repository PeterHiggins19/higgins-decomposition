# JPN_slices — Japan EMBER per-year SVG slices

Per-year SVG slice exports for Japan EMBER electricity generation
(2000–2025), ~3 slices per year (XY / XZ / YZ orthogonal projections).

These were produced by the original Hs pipeline projector. The current
canonical equivalent is the Stage 1 plate book at
[`../../HCI-CNT/conference_demo/cnt_demo/02_per_country/ember_jpn/stage1_ember_jpn.pdf`](../../HCI-CNT/conference_demo/cnt_demo/02_per_country/ember_jpn/stage1_ember_jpn.pdf),
which is hash-chained, auto-titled, and uses the FIXED-scale ILR-Helmert
projection.

The slice files here are kept for lineage.
