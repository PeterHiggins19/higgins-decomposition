# Hs_Direct — direct Higgins instrument outputs

Direct readouts from the Hs instrument on selected datasets. This is
the raw projector / slice / face-export folder.

The 78-file [`JPN_slices/`](JPN_slices/) subfolder contains per-year
SVG slice exports for Japan EMBER electricity generation (2000–2025).

## Status

This folder reflects the original Hs pipeline output format. The
current canonical engine is **CNT 2.0.4** in [`../HCI-CNT/`](../HCI-CNT/);
its plate-time projector is the interactive successor to these slice
files. See:

* [`../HCI-CNT/atlas/plate_time_projector.html`](../HCI-CNT/atlas/plate_time_projector.html)
  — the current 3D interactive projector (barycenter z-spine, valid-known
  connections only)
* [`../HCI-CNT/conference_demo/cnt_demo/02_per_country/ember_jpn/stage1_ember_jpn.pdf`](../HCI-CNT/conference_demo/cnt_demo/02_per_country/ember_jpn/stage1_ember_jpn.pdf)
  — the current Stage 1 paged plate book for Japan

The slice files here remain a valid record of the foundational pipeline
output and are preserved for lineage.
