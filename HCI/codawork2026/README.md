# HCI/codawork2026 — CodaWork 2026 working folder (HCI-side)

Working folder used during the development of the CodaWork 2026
deliverable on the HCI side of the foundational pipeline. Contains
JPN-specific outputs, the cnt_presentation source SVGs, the stage1
plates working folder, and various briefing artefacts.

## Subfolders

| Subfolder | Contents |
|---|---|
| [`cnt_presentation/`](cnt_presentation/) | Source SVGs and PptxGenJS build for the CNT-section slides (see folder README) |
| [`stage1_plates/`](stage1_plates/) | Stage 1 plate working folder for HCI-side experiments |

## Status

This folder predates the CNT-canonical organisation under
[`../../HCI-CNT/`](../../HCI-CNT/). The current canonical CodaWork
demo package is at
[`../../HCI-CNT/conference_demo/`](../../HCI-CNT/conference_demo/),
which is self-contained: engine source, per-country reports, combined
views, calibration, doctrine, talk plan, and the 10-slide PowerPoint
deck.

The material here is preserved for lineage. New CodaWork-related work
goes into `HCI-CNT/conference_demo/`.
